# Milo Platform

Milo is a multi-tenant agentic AI program manager designed for SMEs and solo trade shops.

## Prerequisites
- Python 3.12 (managed via `uv`)
- Node.js 22 LTS (managed via `pnpm`)
- AWS CDK v2

## Quick Start
```bash
make setup
make test-unit
```
